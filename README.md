# Raffle Management System

A full-stack web application for managing online raffles (Chinese auctions).

## Overview
The system provides separate interfaces for administrators, donors, and users,
with secure role-based access and server-side business logic.

## Tech Stack
- Frontend: Angular
- Backend: ASP.NET Core
- Database: SQL Server
